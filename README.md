# DSA-LAB

A collection of **Data Structures and Algorithms implementations** written in **C and C++**.

This repository contains basic implementations of different DSA concepts for learning, practice, and reference purposes.

## 📚 Implemented Topics

### Searching Algorithms
- Linear Search (C++)
- Binary Search (C++)

### Recursion
- Factorial using Recursion (C++)
- Fibonacci Sequence using Recursion (C++)
- Sum of First N Natural Numbers using Recursion (C++)

### Data Structures

#### Stack
- Stack using Array (C)
- Stack using Linked List (C)

#### Queue
- Queue using Array (C)
- Queue using Linked List (C)
- Circular Queue using Array (C)

#### Linked List
- Singly Linked List Implementation (C)

#### Stack Applications
- Infix Expression Evaluation using Stack (C++)

#### Arrays
- Dynamic Array Average Calculation (C++)


## 🛠️ Languages Used

- C
- C++
